# 🎲 骰宝游戏

一个基于iPad的骰宝游戏系统，支持单人和多人模式。

## ✨ 功能特点

### 游戏模式
- **👤 单人模式**：1个庄家 vs 1个闲家
- **👥 多人模式**：1个庄家 vs 多个闲家（2-6人，开发中...）

### 核心功能
- 🎯 实时押注系统
- 💰 筹码管理（10/30/50/100/300/500）
- 📊 自动积分结算
- 📜 20局历史记录
- ✨ 中奖闪烁动画
- 💚 积分增加特效
- 📱 iPad优化界面

### 押注类型
- **大/小**：1赔1
- **双围**（对子）：1赔8
- **双骰组合**：1赔5
- **点数4-17**：不同赔率
- **三围**（豹子）：1赔150
- **全围**（任意豹子）：1赔24
- **单骰**：1赔1

## 🎮 使用方法

### 单人模式
1. 打开`index.html`
2. 选择筹码（默认50）
3. 点击押注区域下注
4. 输入三个骰子点数
5. 点击"确认开奖"
6. 查看中奖结果和积分变化

### 切换模式
1. 点击右上角模式按钮（👤 或 👥）
2. 在弹窗中选择模式
3. 多人模式下选择玩家数量
4. 确认切换

### 操作技巧
- **左键点击**：下注
- **右键点击**：移除筹码
- **长按0.5秒**：清除该区域所有筹码
- **清除按钮**：清除所有押注

## 💾 数据持久化

游戏数据自动保存到浏览器本地存储（localStorage），包括：
- 庄家/闲家余额
- 当前选中筹码
- 历史记录
- 游戏模式设置
